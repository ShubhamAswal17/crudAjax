This project is a Core PHP CRUD application developed using PHP, MySQL, AJAX, jQuery, Bootstrap, HTML, and CSS. The main purpose of the project is to perform Create, Read, Update, and Delete operations without page reloading using AJAX.

Features included in the project:

User-friendly responsive interface using Bootstrap
Insert data into MySQL database using AJAX
Display records dynamically without page refresh
Update existing records using AJAX requests
Delete records instantly with confirmation
Form validation using jQuery and PHP
JSON response handling for AJAX communication
Reusable PHP files for multiple modules
Secure database connection using MySQLi/PDO
Fast and smooth user experience through asynchronous requests

Technologies Used:

PHP
MySQL
AJAX
jQuery
Bootstrap
HTML5
CSS3
JavaScript

This project helped in understanding database operations, asynchronous data handling, form processing, and dynamic web application development using Core PHP and AJAX.
